# Groq JS Chatbot

A simple terminal chatbot in JavaScript that calls the Groq API.

## Prerequisites

- Node.js 18+
- A Groq API key

## Setup

1. Install dependencies:

```bash
npm install
```

## Run

```bash
npm start
```

Type `exit` to quit.

## Notes

- Default model is `llama-3.1-8b-instant`.
- You can change model in `.env` using `GROQ_MODEL`.
